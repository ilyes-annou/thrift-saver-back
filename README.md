# Thrift Saver Fromnt Mobile

>Backend part of Thrift Saver App. 

## Installation

To install this project on your computer, download or clone the repo and open it on VSCode.
Run npm install


## Endpoints

### Spending

GET /spending

GET /spending/:id

POST /spending

PUT /spending/:id

DELETE /spending/:id

### User

GET /user

GET /user/:id/spending

GET /user/:id

POST /user

POST /login

PUT /user/:id

DELETE /user/:id
